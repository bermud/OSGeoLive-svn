Arramagong GIS Virtual Machine and ISO-Image Downloads

Try out fully-operational versions of popular Free Geospatial Software without the need to install a thing! Everything is set up for you ready to go: just reboot into it, glance at the help pages, and start using it. Your existing hard drive and operating system will not be modified in any way, everything happens in temporary system memory.

We assume that Alpha, MIPS, SPARC, etc., users can teach us more about building software than we can them, and do not need our advice.
