Available ISO-Images and Virtual Machines:

Try out fully-operational versions of popular Free Geospatial Software on any* computer without the need to install a thing! Everything is set up for you ready to go: just reboot into it, glance at the help pages on the desktop, and start using it. Your existing hard drive and operating system will not be modified in any way, everything happens in temporary system memory. As such it will run a bit slower than the "real thing", and you may want to grab a USB flash drive to save or transfer files with. You can load and boot a live-image completely from a USB stick too if you prefer.

  • What is a Live ISO image? See the Wikipedia entry
  • What is a Virtual Machine? See the Wikipedia entry

[*] Minimum suggested resources: 512MB RAM, 1GHz CPU. No hard drive required. Sorry ARM users, by "any" computer we mean any i686 compatible. We assume that Alpha, MIPS, SPARC, etc., users can teach us more about building software than we can them, and do not need our advice.

   # recombine
   cat arramagong-livedvd-2.0.1-Final.iso.0 \

       > arramagong-livedvd-2.0.1-Final.iso

   md5sum arramagong-livedvd-2.0.1-Final.iso
If the download is stopped for some reason part way through, continue without restarting from the beginning by making sure the filename is unaltered (rename if needed) and running "wget -c {URL}" from the directory that the partially downloaded file is in.
