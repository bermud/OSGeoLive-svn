OSGeo Live GIS Virtual Machine and ISO-Image Downloads

An entire Geospatial software stack and operating system available on:
  • Bootable DVD
  • Bootable USB stick
  • Virtual Machine application
  • and even Online
Try out fully-operational versions of over 30 popular Free Geospatial Software programs without having to install a thing! Everything is set up for you ready to go: just reboot into it, glance at the help pages, and start exploring. Your existing hard drive and operating system will not be modified in any way, everything happens in temporary system memory. A collection of tutorials and sample data is included.

Available downloads are:
  • Image files: Due to the large size of the ISO and VM files, images have been split into multiple parts which will need to be put back together again. Refer to the Quick Start guide for details.
  • via BitTorrent (howto)

[*] Minimum suggested system resources: 512MB RAM, 1GHz i688 compatible CPU. No hard drive required. Mac users will benefit from a USB mouse. Persistent USB stick install requires a minimum 4gb drive.
