The set of scripts in this directory will make a GIS Virtual Machine
from a base XUbuntu system.

Running:
=======

sudo ./bin/main.sh

File Structure:
==============

gisvm/
     /master.sh # Call all the other scripts
     /core.sh # Download, and install all core files and set up config files
     /project1.sh # Download, and install all files for project1
     /project2.sh # Download, and install all files for project2
