Μεταφόρτωση - Download
================================================================================

Η τελευταία σταθερή έκδοση του OSGeo-Live περιλαμβάνει:

.. image:: ../images/download_buttons/download-dvd.png
  :alt: Κατεβάστε το αρχείο iso συμπεριλαμβανομένων προγραμμάτων εγκατάστασης για Windows
  :align: left
  :target: http://download.osgeo.org/livedvd/release/5.0/osgeolive-5.0.iso

`osgeo-live-5.0.iso <http://download.osgeo.org/livedvd/release/5.0/osgeolive-5.0.iso>`_ : |osgeolive-iso-size|

Ένα DVD αυτόματης εκκίνησης βασισμένο στην διανομή Xubuntu, το οποίο περιλαμβάνει αρχεία για τα λειτουργικά Windows και Mac. Μπορεί επιπλέον να αντιγραφεί σε USB αυτόματης εκκίνησης μεγέθους 8 GB. Επιβεβαιώστε το κατεβασμένο αρχείο σας με το md5sum checksum `(osgeo-live-5.0.iso.md5) <http://download.osgeo.org/livedvd/release/5.0/osgeolive-5.0.iso.md5>`_.

.. image:: ../images/download_buttons/download-mini.png
  :alt: Κατεβάστε το αρχείο μεγέθους 3.3 GB χωρίς τα αρχεία για Windows και Mac
  :align: left
  :target: http://download.osgeo.org/livedvd/release/5.0/osgeolive-5.0-mini.iso

`osgeo-live-mini-5.0.iso <http://download.osgeo.org/livedvd/release/5.0/osgeolive-5.0-mini.iso>`_ : |osgeolive-iso-mini-size|

Το ίδιο με το osgeo-live-5.0.iso, με τη διαφορά ότι δεν περιλαμβάνει τις εγκαταστάσεις για  Windows και Mac και μπορεί να αντιγραφεί σε USB αυτόματης εκκίνησης  μεγέθους 4 GB. Επιβεβαιώστε το κατεβασμένο αρχείο σας με το md5sum checksum `(osgeo-live-mini-5.0.iso.md5) <http://download.osgeo.org/livedvd/release/5.0/osgeolive-5.0.iso.md5>`_.

.. image:: ../images/download_buttons/download-vm.png
  :alt: Κατεβάστε το αρχείο 7-zip μεγέθους 2.6 GB μιας εικονικής μηχανής χωρίς αρχεία εγκατάστασης Windows και Mac
  :align: left
  :target: http://download.osgeo.org/livedvd/release/5.0/osgeo-live-5.0-vm.7z

`osgeo-live-5.0-vm.7z <http://download.osgeo.org/livedvd/release/5.0/osgeo-live-5.0-vm.7z>`_ : |osgeolive-vm-7z-size|

Εικονική μηχανή (.vmdk) κατάλληλη για χρήση με τα VirtualBox, VMWare, KVM και άλλα λογισμικά εικονικών μηχανών. Δεν περιλαμβάνει προγράμματα εγκατάστασης για windows και mac. Επιβεβαιώστε το αρχείο με το md5sum checksum `(osgeo-live-vm-5.0.7z.md5) <http://download.osgeo.org/livedvd/release/5.0/osgeo-live-5.0.7z.md5>`_.

Για μια πλήρη λίστα από εκδόσεις αρχείων, επιλογές μεταφόρτωσης και εναλλακτικούς παρόχους μεταφόρτωσης επισκεφθείτε: http://download.osgeo.org/livedvd

[*] `Ελλάχιστοι προτεινόμενοι πόροι συστήματος: μνήμη 768MB RAM, συμβατός επεξεργαστής 1GHz i688. Δεν απαιτείται σκληρός δίσκος. Οι χρήστες Mac θα εποφεληθούν από ένα ποντίκι USB.`
