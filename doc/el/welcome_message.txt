Καλώς ήρθατε στο δοκιμαστικό GIS DVD του OSGeo.


Επιλέξτε το εικονίδιο Help στην επιφάνεια εργασίας για πληροφορίες σχετικά με το δοκιμαστικό DVD και την εικονική μηχανή virtual-machine, που περιλαμβάνει μια σύνοψη των συμπεριλαμβανομένων γεωχωρικών λογισμικών και μια σειρά από εγχειρίδια ταχείας εκμάθησης.

Για αλλαγή της γλώσσας, αποσυνδεθείτε (από το μενού εφαρμογών) και στο κάτω μέρος της οθόνης σύνδεσης θα βρείτε το μενού της γλώσσας. 
Με το που συνδεθείτε ξανά (ο κωδικός πρόσβασης είναι "user"), υπάρχει ένα εικονιίδιο σημαίας στο πάνω μέρος της οθόνης που σας επιτρέπει να διαλέξετε την κατάσταση γλώσσας του πληκτρολογίου. Μπορείτε επίσης να θέσετε τη γλώσσα επιλογής σας από το μενού Applications > System > Language Support.

Τα προγράμματα εγκατάστασης για τα λειτουργικά συστήματα Macintosh OSX και Microsoft Windows μπορούν να βρεθούν στους φακέλους MacInstallers και WindowsInstallers.

Οι κωδικοί ασφαλείας για τις διάφορες εφαρμογές βρίσκονται στο αρχείο passwords.txt στην επιφάνεια εργασίας.

Εαν χρειαστείτε να συνδεθείτε σαν χρήστης "root" (ο διαχειριστής στα συστήματα UNIX) ανοίξτε ένα τερματικό και πληκτρολογήστε "sudo su", και στη συνέχεια επιλέξτε "user" σαν κωδικό πρόσβασης.

Οι συχνότερες ερωτήσεις (FAQ) μπορούν να βρεθούν στον παρακάτω σύνδεσμο:
