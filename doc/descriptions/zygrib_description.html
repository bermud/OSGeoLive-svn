zyGrib

About

zyGrib is a program to download and visualize weather forecast data in the standard GRIB distribution format.

Homepage

http://www.zygrib.org

License

GNU General Public License (GPL) version 3

Running

This application, written with the Qt toolkit, enables:
  • Visualisation of meteorologic data from files in GRIB Format 1
  • Automatic GRIB data download
  • Automatic Download from IAC (fleetcode) Data
