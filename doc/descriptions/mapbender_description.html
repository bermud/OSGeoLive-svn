Mapbender

About

Mapbender is a web based geoportal framework. Mapbender is implementing the publish/register, find, bind, execute paradigm for geospatial data.

Documentation

Running

To run Mapbender on the Live DVD, click on the Mapbender link on the desktop or open a browser and go to http://localhost/mapbender/. Then you can login to Mapbender with user root and password root.
