OpenCMN

About

OpenCPN is a free software project to create a concise chart plotter and navigation software for use as an underway or planning tool.

Current features: 

* GPS/GPSD Postition Input
* BSB Raster Chart Display
* S57 Vector ENC Chart Display
* C-MAP Vector Chart Display (CM93/2)
* AIS Input Decoding
* Waypoint / Autopilot Navigation
* International Language Support
* Cross-platform support
* Grib Weather Overlay

Running

Setting up your charts

Before running OpenCPN on the LiveDVD you should activate the sample data. To do this open a Terminal from the main Applications menu and run "opencpn_noaa_agreement.sh" at the command prompt.

You can download additional US nautical charts from NOAA at http://www.charts.noaa.gov/

Setting up your GPS

If you have a GPS connected you will want to start the gpsd service. To do that run: 

  sudo dpkg-reconfigure gpsd
and 
  sudo /etc/init.d/gpsd start
BlueTooth GPS users should take special care to read up about the no-probe option. (for this reason we have not started gpsd automatically)

• 'xgps' is a good program to check that Gpsd can see your GPS ok.
