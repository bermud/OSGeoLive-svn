Download

The latest stable release of OSGeo-Live contains:
  • osgeo-live-4.0.iso, 4.3 Gig: An xubuntu based bootable DVD, which also includes windows and mac installers. It can also be copied to an 8 Gig bootable USB.
  • osgeo-live-mini-4.0.iso: 3.2 Gig Same as osgeo-live-4.0.iso, except it doesn't contain windows and mac installers and can be copied onto a 4 Gig bootable USB.
  • osgeo-live-mini-vm-4.0.tar.gz: 3.8 Gig A .vmdk image ready to be loaded into a virtual machine like VirtualBox or VMWare Player.
Download from: http://download.osgeo.org/livedvd

[*] Minimum suggested system resources: 512MB RAM, 1GHz i688 compatible CPU. No hard drive required. Mac users will benefit from a USB mouse. Persistent USB stick install requires a minimum 4gb drive.

Quick Starts

  • Run the OSGeo-Live DVD
  • Load OSGeo-Live ISO or DVD in VirtualBox Virtual Machine
  • Instal OSGeo-Live onto a Hard Disk or Virtual Machine
  • Create an OSGeo-Live bootable USB flash drive
