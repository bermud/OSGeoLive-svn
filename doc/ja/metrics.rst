|osgeo-live-version| プロジェクトメトリクス
================================================================================

プロジェクトのコードリポジトリ `OpenHUB <https://www.openhub.net/>`_ によるメトリクス。
これらのメトリクスにはいくつかの欠点 (リポジトリの移動などによる) があることに注意する必要があります。
また、プロジェクト開発については断片的な情報しかありません。
一部の小さな変更を除けば、パッケージとライブラリは完成度の高いレベルにあります。

メトリクスを見るためにはインターネット接続と　Javascript　の実行をを許可する必要があります。

.. raw:: html

  <script type="text/javascript" src="http://www.openhub.net/p/OSGeo-Live/widgets/project_factoids_stats.js"></script><br/>
