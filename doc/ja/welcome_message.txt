OSGeo Live GIS DVD へようこそ！


Live ディスクと仮想マシンについて知りたい方はデスクトップの
ヘルプアイコンをご覧下さい。
バンドルしている地理空間パッケージの情報や、
それぞれのクィックスタート＝短い手引きを収録しています。


言語を変更するには、アプリケーションメニューからログアウトしてください。
user を選択すると画面下部に言語の選択メニューが表示されますので、
選択して最ログインをしてください。（パスワードは user です）
上部のメニューバーにある旗のアイコンから、キーボード配列を
変更することもできます。また、メニューから アプリケーション
＞システム＞言語サポート　を選択することでも言語やキーボード配列
を変更できます。


MacInstallers や WindowsInstallers フォルダ内には、
Mac OS X や Microsoft Windows 用のインストーラを格納しています。


パスワードやその他の情報は、
デスクトップの passwords.txt に記載しています。


rootユーザ(UNIX のスーパーユーザ)になる必要がある場合は、
ターミナルを開き、"sudo su" の後、パスワード"user" 
を入力してください。


最新のFAQは以下の URL でご覧頂けます。
http://wiki.osgeo.org/wiki/Live_GIS_Disc_FAQ
