Selamat Datang di OSGEO Live GIS DVD.

Gunakan ikon penunjuk jalan untuk bantuan dari Desktop serta informasi      
mengenai Live-DVD dan mesin virtual, termasuk didalamnya penjelasan
singkat dari paket aplikasi geopasial yang tergabung dan beberapa
petunjuk panduan singkat.

Untuk mengubah bahasa, pilih Keluar (Menu Aplikasi) dan dibagian bawah
jendela login terdapat menu drop-down bahasa. Setelah anda masuk
kembali (username dan password keduanya adalah "user"), terdapat ikon
bendera pada baris menu bagian atas yang memungkinkan anda untuk
menentukan tata letak papan ketik. Anda juga bisa mengatur melalu menu
dengan menuju Aplikasi > Pengaturan > Dukungan Bahasa.

Jika anda memiliki edisi DVD lengkap, beragam instalasi program untuk
Macintosh dan Microsoft Windows dapat ditemukan dalam direktori
MacInstallers dan WindowsInstallers; lihat halaman bantuan utama.

Password untuk beberapa hal lainnya dapat ditemukan pada file
passwords.txt di desktop.

Jika anda menginginkan untuk menjadi "root" (UNIX super-user) buka
Terminal dan ketikkan "sudo su", kemudian masukkan "user" sebagai
password jika diminta.

Daftar pertanyaan yang sering ditanyakan (FAQ) dapat dilihat disini :
http://wiki.osgeo.org/wiki/Live_GIS_Disc_FAQ
