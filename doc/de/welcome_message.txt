Willkommen auf der OSGeo Live GIS DVD.


Auf dem Desktop finden Sie einen Hilfe Button, über den
Informationen über die Live DVD und virtuelle Maschine abgerufen
werden können. Außerdem finden Sie hier Informationen zu den
Softwareprojekten und eine Reihe Tutorials.

Um die Sprache zu ändern, müssen Sie sich abmelden. Unterhalb des
Anmeldefensters finden Sie das Menü zur Sprachauswahl.  Nach dem
erneuten Anmelden (das Paßwort ist "user") finden Sie ein
Flaggensymbol im Menü, über das die Tastaturbelegung ausgewählt
werden kann. Diese Tastaturbelegung kann auch über die
Systemsteuerung über Anwendungen > System > Language Support
erfolgen.

Installer für Mac OS X und Microsoft Windows finden sich in
den Verzeichnissen MacInstallers und WindowsInstallers

Verschiedene Paßworte finden sich auf dem Desktop in der Datei
passwords.txt.

Wenn Sie "root" (UNIX super-user) werden wollen, öffnen Sie einen
Terminal und setzen Sie den Befehl "sudo su" ab. Geben Sie
anschließend das Paßwort "user" ein.

Das FAQ finden Sie hier:
