Format¶

The format is very simple. Each line contains 2 items: an alias and the name/path of the font separated by white space. The alias is simply the name you refer to the font as in your Mapfile (eg. times-bold). The name is the actual name of the TrueType file. If not full path then it is interpreted as relative to the location of the fontset. Here’s the fontset I use (the font.list file and all .ttf files are stored in the same sub-directory).

Note

Aliases are case sensitive. Excellent reference information about the TrueType format and online font resources is available from the FreeType.
