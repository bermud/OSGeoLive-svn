Welcome to MapServer¶

MapServer is an Open Source platform for publishing spatial data and interactive mapping applications to the web. Originally developed in the mid-1990’s at the University of Minnesota, MapServer is released under an MIT-style license, and runs on all major platforms (Windows, Linux, Mac OS X).

The MapServer Project Steering Committee (PSC), sanctioned by OSGeo, manages and administers the project which is maintained, improved, and supported by developers from around the world. MapServer is not a full-featured GIS system, nor does it aspire to be.

Announcements¶

7-24-2009 - MapServer 5.4.2 is released (also 5.2.3 and 4.10.5)¶

MapServer 5.4.2 has been released, which includes a security fix. All users of pre-5.4 releases should upgrade to 5.4.2 as soon as possible. See the Changelog for more details and head to Download to obtain a copy.

4-22-2009 - MapServer 5.4 is released¶

MapServer 5.4 has been released. See the Changelog for more details and head to Download to obtain a copy.

3-26-2009 - MapServer 5.2.2 and 4.10.4 security fixes are released¶

MapServer 5.2.2 and 4.10.4 have been released, containing important security fixes. See the Announcement for more details and head to Download to obtain copies.

3-8-2009 - MapServer PSC Attend Toronto Code Sprint¶

Members of the MapServer Project Steering Committee attend the Toronto Code Sprint event, an international meeting of GDAL, PostGIS, OpenLayers, and MapServer development teams.

12-30-2008 - New MapServer website is now live¶

The new, improved MapServer website is now live! Welcome to the new MapServer website. Some of the major improvements include:

  • An updated Glossary
  • A significantly improved An Introduction to MapServer
  • All documentation is now available in a single, 600+ page PDF
  • More internal document referencing, better styling, and mapfile styling,
  • Simpler organizational structure

12-16-2008 - MapServer Graduates OSGeo Incubation¶

MapServer recently graduated OSGeo’s Incubation process. Incubation includes ensuring items like code provenance and community contribution. See the announcement for more details.
